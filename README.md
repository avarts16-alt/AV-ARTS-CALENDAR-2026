# AV ARTS Calendar 2026 — App Icon Edition

This repository hosts a web app to view your 12-month calendar PDF.

## How to Use
1. Upload `AV ARTS_CALENDAR_2026.pdf` from your computer.
2. View each month page-by-page.
3. Installable as an app (Add to Home Screen).

## Files Required
- index.html
- manifest.json
- av.jpg (logo)
- AV ARTS_CALENDAR_2026.pdf
- README.md

## Deploy to GitHub Pages
1. Create a public repo (e.g. avarts-calendar-2026).
2. Upload all files.
3. Go to Settings → Pages → Branch: `main`, Folder: `/ (root)`.
4. Save. Your app will be live soon!
